# Lottie Compressor

## Dependencies

- PNGQuant
- oxipng
- ujson

## Process

- Search for data img
- Extract
- Compress with pngquant
- Optimize with oxi png
- insert in json
- optimize json with ujson

## Usage

`python3 compressor.py`

When prompted provide path to folder or file
